<h1>modernJs</h1>

<h2>브라우저에서 할 수 있는 일</h2>
<ol>
    <li>새로운 html을 추가, 변경</li>
    <li>이벤트, 즉 사용자의 행동애 반응</li>
    <li>네트워크를 통해 서버에 요청, 파일 다운 업로드 등</li>
    <li>쿠키설정, 사용자와 소통</li>
    <li>클라이언트에 데이터 저장 - 로컬 스토리지</li>
</ol>

<h2>브라우저에서 할 수 없는 일</h2>
<ol>
    <li>디스크에 저장된 파일을 읽거나 쓰기, 복사, 실행 등</li>
    <li>동일 출처 정책 : 각 페이지 끼리 정보를 알 수 없으나 js를 사용해서 열리면 가능하다 하지만 도메인, 프로토콜, 포트가 다르면 불가</li>
    <li>타 사이트나 도메인에서 데이터를 받아오는 것, 가능 하다해도 명확한 승인이 필요</li>
</ol>

<h2>js의 강점</h2>
<ol>
    <li>html/css와 통합이 가능</li>
    <li>간단한 일은 간단히 처리</li>
    <li>모든 브라우저에서 지원 및 기본 언어로 사용</li>
</ol>

<h2>또 다른 js</h2>

</ol>
